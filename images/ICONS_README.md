# アイコンについて

この拡張機能を完全に動作させるには、以下のサイズのアイコン画像が必要です：

- `icon16.png` - 16x16ピクセル
- `icon48.png` - 48x48ピクセル
- `icon128.png` - 128x128ピクセル

## アイコンの作成方法

### オプション1: オンラインツールを使用

1. [Canva](https://www.canva.com/)や[Figma](https://www.figma.com/)などのデザインツールを使用
2. カレンダーとAIをイメージしたアイコンをデザイン
3. 各サイズでエクスポート

### オプション2: シンプルな単色アイコン

開発中は、以下のような簡単な方法で仮アイコンを作成できます：

1. 任意の画像編集ソフトを使用
2. 背景色を選択（例：紫 #667eea）
3. 「📅」や「AI」などのテキストを配置
4. 各サイズで保存

### オプション3: アイコンジェネレーターを使用

以下のようなツールで自動生成できます：

- [Icon Generator](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
- [Favicon Generator](https://realfavicongenerator.net/)

## 推奨デザイン

- カレンダーのシンボル（📅）
- AIや脳のアイコン（🤖、🧠）
- グラデーション配色（紫系統が推奨）
- シンプルで認識しやすいデザイン

## 配置

作成したアイコンをこのディレクトリ（`images/`）に配置してください：

```
images/
├── icon16.png
├── icon48.png
└── icon128.png
```

アイコンがない場合、Chrome拡張機能の読み込み時に警告が表示されますが、機能自体は動作します。
